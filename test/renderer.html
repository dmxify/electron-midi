Welcome to dmxify/electron-midi

npm package github repo

This zero-dependancy package uses the Web MIDI API which is available to Electron's renderer processes.

Try the following:

  • Physically attach / detach a MIDI device from your computer.
    The drop-down boxes should update accordingly (provided the device drivers are installed)
  • Send midi messages to and from a midi-device
    Have a look in devtools. you can also set listeners
  • "Learn" input by pressing the button below
    Uses electronMidi.learn().then(response) - an ES6 promise method
